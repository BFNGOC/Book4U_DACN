🚀 Bắt Đầu Sử Dụng

1. Setup Server

cd Server npm install multer npm start

2. Setup Client

cd Client/Book4U npm install npm run dev

3. Test Upload

node test-chat-file-setup.js

4. Gửi File Trong Chat

  1. Mở browser: http://localhost:5173
  2. Đăng nhập
  3. Mở Chat
  4. Click nút 📎 (Upload)
  5. Chọn file
  6. Click "Gửi file"
